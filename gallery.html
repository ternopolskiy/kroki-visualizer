О Kroki API

Kroki API предоставляет унифицированный интерфейс для создания диаграмм с использованием различных синтаксисов. Вы можете легко интегрировать эти диаграммы в свои проекты, документацию или презентации.

Формат URL для API: https://kroki.io/{diagram_type}/{output_format}/{encoded_diagram_source}

Где:

  • {diagram_type} — тип диаграммы (plantuml, mermaid, graphviz и т.д.)
  • {output_format} — формат вывода (svg, png, jpeg, pdf)
  • {encoded_diagram_source} — исходный код диаграммы, закодированный в base64
