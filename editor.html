Редактор диаграмм

Создавайте и визуализируйте диаграммы с помощью API Kroki.io

Советы по использованию

  • Выберите тип диаграммы из выпадающего списка
  • Введите код диаграммы в текстовое поле
  • Нажмите кнопку "Визуализировать" для отображения результата
  • Используйте кнопку "Сохранить" для скачивания диаграммы

О Kroki API

Kroki API позволяет создавать диаграммы с помощью текстового описания. Формат URL для API:

https://kroki.io/{diagram_type}/{output_format}/{encoded_diagram_source}

Примеры синтаксиса

PlantUML
@startuml
actor User
User -> System: Request
System --> User: Response
@enduml
Mermaid
graph TD
A[Начало] --> B{Условие?}
B -->|Да| C[Процесс 1]
B -->|Нет| D[Процесс 2]
GraphViz
digraph G {
  rankdir=LR;
  A -> B -> C;
  B -> D;
}
