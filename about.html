О проекте Kroki Визуализатор

Kroki Визуализатор — это веб-приложение, которое предоставляет удобный интерфейс для работы с API сервиса Kroki.io. Наш проект позволяет создавать, редактировать и экспортировать различные типы диаграмм с помощью простого текстового описания.

Что такое Kroki.io?

Kroki.io — это сервис, который предоставляет унифицированный API для создания диаграмм с использованием различных инструментов и синтаксисов. Kroki поддерживает множество форматов диаграмм, включая PlantUML, Mermaid, GraphViz, BPMN, C4 и многие другие.

Поддерживаемые типы диаграмм

В нашем приложении вы можете создавать следующие типы диаграмм:

  • PlantUML — язык для создания UML-диаграмм (диаграммы классов, последовательностей, состояний и др.)
  • Mermaid — простой язык разметки для создания диаграмм из текстовых описаний
  • GraphViz — инструмент для визуализации структурной информации в виде графов и сетей
  • C4 с PlantUML — модель для визуализации архитектуры программного обеспечения
  • BPMN — нотация моделирования бизнес-процессов

Форматы экспорта

Вы можете экспортировать созданные диаграммы в следующих форматах:

  • SVG — масштабируемая векторная графика, идеально подходит для веб-страниц
  • PNG — растровое изображение с поддержкой прозрачности
  • JPEG — растровое изображение с высокой степенью сжатия
  • PDF — формат для печати и документации

Как использовать

Использование нашего сервиса очень просто:

  1. Перейдите на страницу Редактор
  2. Выберите тип диаграммы из выпадающего списка
  3. Введите код диаграммы в текстовое поле (или используйте предложенный пример)
  4. Нажмите кнопку «Визуализировать» для отображения результата
  5. При необходимости выберите формат экспорта и нажмите «Сохранить»

Примеры

Если вы не знаете, с чего начать, посетите нашу Галерею, где вы найдете примеры различных типов диаграмм. Вы можете просмотреть код любого примера и использовать его как основу для своей диаграммы.

Технологии

Проект разработан с использованием следующих технологий:

  • HTML5 для структуры страниц
  • CSS3 для стилизации и создания адаптивного дизайна
  • JavaScript для интерактивности и работы с API
  • API Kroki.io для генерации диаграмм
